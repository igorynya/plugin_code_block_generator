## Описание плагина:
<!-- Плагин работает следующим образом: пользователь выбирает, какой блок код ему нужен и происходит генерация этого блока. Выбрать блок кода можно двумя способами: нажать п.к.м. и выбрать "Generate code block" и в выпаающем списке выбрать необходимый блок, или сочетанием клавим cmd+option+V открыть список блоков и выбрать необходимый. -->
Плагин вставляет необходимый пользователю для решения данной задачи блок кода

## Возможности плагина:
1) С помощью сочетания клавиш cmd+option+V открыть списко выбора блоков
2) Нажать п.к.м., а потмо выбрать в появившемся окне "Generate code block" и открыть список выбора блоков
3) В открывшемся списке можно выбрать необходимый из представленных блок кода. Плагин умеет вставлять следующие блоки: if, while, for, try-catch, switch (только для языка C++)
4) Даллее программа вставляет необходимый блок кода и двигает курсор в то место, которое пользователь должен изменить в зависимости от решаемой задачи
